# CLIPVision TT-NN Implementation

This directory contains an implementation of the CLIPVision model using TT-NN.

## Quick Start

```bash
# To sparse clone this dir
git clone --filter=blob:none --no-checkout https://github.com/tenstorrent/tt-xla tt-xla-clipvision
cd tt-xla-clipvision
git sparse-checkout set clipvision_opt2_b2
git checkout svuckovic/clipvision-demo
cd clipvision_opt2_b2

# To run
python main.py
```

This will run the model for N iterations, printing FPS (frames per second) and comparing the output result to PyTorch implementation (PCC).

## About This Code

This code was generated by running `examples/pytorch/clipvision.py` and then editing it slightly for better readability.

The compiler has already sharded some ops for improved performance: by default, throughput varies from **50 to 70 FPS**.

## Directory Structure

```
clipvision_opt2_b2/
- main.py           # Main entry point - model execution and benchmarking
- model.py          # Neural network layer implementations
- consteval.py      # Parameter transformation functions
- loader.py         # Parameter loading utilities
- tensors/          # Serialized TT-NN parameter tensors (*.tensorbin files)
```
**Note:** The batch size for this implementation is set to **2**.
